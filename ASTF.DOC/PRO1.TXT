
	Система разработки программного обеспечения
			 АстроФОРТ+


     Система АстроФОРТ	является реализацией стандарта языка
ФОРТ-83 для персональных компьютеров, совместимых с машинами
семейства  IBM	 PC,   оснащенных   микропроцессорами	типа
i8086/i8088.  Система  АстроФОРТ  может  использоваться  для
разработки программного  обеспечения широкого  назначения  -
операционных  систем,	автоматизированных   рабочих   мест,
диалоговых  информационно-справочных   систем,	 игровых   и
обучающих программ и т.д.
     В комплект  системы АстроФОРТ+ входит система АстроФОРТ
и набор  сервисных  пакетов,  предоставляющих  пользователям
системы дополнительные возможности:
  -  пакет графических подпрограмм для графики высокого раз-
     решения (640x200  точек, 2 цвета, испльзуется с   CGA и
     EGA);
  -  пакет  графических  подпрограмм  для  графики  среднего
     разрешения (320x200  точек, 4 цвета, используется с CGA
     и EGA);
  -  пакет  графических   подпрограмм  для  графики  низкого
     разрешения  (160x100  точек,  16  цветов,	используется
     только с CGA);
  -  пакет  графических  подпрограмм  для  графики  высокого
     разрешения  (640x350  точек,  16  цветов,	используется
     только с EGA);
  -  пакет генерации звуковых сигналов и музыкальных мелодий
     в фоновом режиме;
   -   пакет поддержки вещественной арифметики с использова-
     нием сопроцессора i8087;
   -  пакет поддержки вещественной арифметики с эмуляцией на
     основном процессоре i8086/i8088;
  -  пакет для	автоматизированного построения пользователь-
     ских "меню",  использующих интерфейс  с  клавиатурой  и
     графическим манипулятором;
  -  пакет для	детрансляции ФОРТ-определений с возможностью
     записи детранслированных текстов в файл.
