                   ==============================
                      СПРАВОЧНИК ПО ФУНКЦИЯМ
                   ==============================
      
      
1.  pcxBeginPaletteGrab  Начинает захват или управление палитрой EGA,
                         для дальнейшего сохранения палитры
      
2.  pcxBufferDisplay     Показывает изображение из буфера
      
3.  pcxBufferFile        Сохраняет буфер изображения в файл
    
4.  pcxBufferPrint       Печатает буфер изображения PCX
      
5.  pcxBufferText        Загружает текстовой экран обратно в дисплей
      
6.  pcxBufferVirtual     Расширяет изображение PCX в виртуальный буфер
      
7.  pcxCreateVirtual     Создает виртуальный буфер
      
8.  pcxDecodePalette     Декодирует палитру PCX в палитру BIOS
      
9.  pcxDestroyVirtual    Уничтожает созданный виртуальный буфер
      
10. pcxDisplayBuffer     Сохраняет  дисплей в буфер изображения
      
11. pcxDisplayFile       Сохраняет дисплей в файл PCX
      
12. pcxDisplayPrint      Печатает дисплей на принтере, указанном
                         pcxSetPrinter
      
13. pcxEMInstalled       Проверяет, если ли расширенная память EMS
      
14. pcxEncodePalette     Кодирует палитру BIOS в палитру PCX
      
15. pcxEndPaletteGrab    Заканчивает управление палитрой EGA, после
                         функции pcxBeginPaletteGrab
      
16. pcxFileBuffer        Загружает файл PCX в буфер изображения
      
17. pcxFileDisplay       Загружает файл PCX непосредственно в дисплей
      
18. pcxFilePrint         Печатает фал изображения PCX
      
19. pcxFileVirtual       Расширяет файл PCX в витруальный буфер
      
20. pcxGetBufferHeader   Читает заголовок PCX из буфера изображения
      
21. pcxGetBufferPalette  Читает палитру из буфера изображения
      
22. pcxGetBufferType     Читает тип дисплея буфера изображения
      
23. pcxGetDisplay        Читает текущий тип дисплея, установленный
                         ранее при помощи функции pcxSetDisplay
      
24. pcxGetDisplayInfo    Читает информацию по типу дисплея
      
25. pcxGetDisplayPalette Читает текущую палитру дисплея
                      
26. pcxGetFileHeader     Читает заголовок PCX из файла PCX
      
27. pcxGetFilePalette    Читает палитру из файла PCX
      
28. pcxGetFileType       Читает тип дисплея файла PCX
      
29. pcxGetImage          Читает изображение из точного адеса и
                         использует логические операции
      
30. pcxGetLibHeader      Читает заголовок PCX из файла PCX в
                         библиотеке изображений
      
31. pcxGetLibPalette     Читает палитру из файла PCX в библиотеке
                         изображений
      
32. pcxGetLibType        Читает тип дисплея файла PCX в библиотеке
                         изображений
      
33. pcxGetPage           Читает текущую страницу дисплея
      
34. pcxGetVersion        Читает текущую версию пакета PCX
      
35. pcxLibBuffer         Загружает файл PCX из библиотеки изображений
                         в буфер изображения
        
36. pcxLibDisplay        Загружает файл PCX из бмиблиотеки изображений
                         в дисплей
      
37. pcxLibPrint          Печатает изображение библиотеки PCX
      
38. pcxLibSearch         Ищет данное изображение в библиотеке
      
39. pcxLibVirtual        Расширяет изображение библиотеки PCX в
                         виртуальный буфер
      
40. pcxmodeCheck         Включает и выключает проверку режима в
                         программах пакета
      
41. pcxPutImage          Выводит изображение по точному адресу и
                         использует логические операции
      
42. pcxQueryChipset      Запрашивает текущий набор чипов VGA
      
43. pcxSetBuffer         Устанавливает внутрений буфер изображения
                         пакета как больший, определенный пользователем
                         буфер
      
44. pcxSetBufferHeader   Устанавливает новый заголовок PCX для буфера
                         изображения
      
45. pcxSetBufferPalette  Устанавливает новую палитру для буфера
                         зображения
      
46. pcxSetDisplay        Устанавливает тип дисплея для всех будущих
                         версий пакета (должна быть вызвана по крайней
                         мере один раз)
      
47. pcxSetDisplayType    Устанавливает новую палитру для дисплея
      
48. pcxSetFileHeader     Устанавливает новый заголовок PCX
                         для файла PCX
      
49. pcxSetFilePalette    Устанавливает новую палитру для файла PCX
      
50. pcxSetHerc           Конфигурирует адаптер "Геркулес"
      
51. pcxSetMode           Изменяет режим дисплея на текстовой (TEXT)
                         или графический (GRAPHICS)
      
52. pcxSetPage           Устанавливает текущую страницу дисплея
      
53. pcxSetPrinter        Устанавливает и конфигурирует текущее
                         устройство печати
      
54. pcxTextBuffer        Сохраняет текстовой экран в буфер
      
55. pcxVerifyBuffer      Проверяет, является ли буфер настоящим
                         буфером изображения PCX
      
56. pcxVerifyFile        Проверяет, является ли файл настоящим
                         файлом PCX
      
57. pcxVideoInfo         Читает информацию по текущей видео-аппаратуре
      
58. pcxVirtualDisplay    Загружает фиртуальный буфер в дисплей
      
59. pcxVirtualPrint      Печатает виртуальный буфер на принтере,
                         указанном функцией pcxSetPrinter
      
60. pcxVirtualScroll     Показывает виртуальный буфер и разрешает
                         прокрутку
       
61. pcxVirtualSize       Подсчитывает объем памяти, необходимый для
                         виртуального изображения
