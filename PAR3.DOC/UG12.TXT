                                                    
                                                    Г Л А В А  12
-----------------------------------------------------------------
               
                        H E L P  (ПОМОЩЬ)
     
-----------------------------------------------------------------
                        
     
     Нажатие клавиши {F1} или выбор услуги Help (помощь) из любо-
го меню системы Paradox позволяет вам сразу же получить на экране
информацию, поясняющую любое действие, которое вы осуществляете в
данный момент.
     Если экран помощи не содержит точной информации, которая вам
необходима, вы можете  воспользоваться  индексом  помощи  системы
Paradox для  нахождения  нужного  экрана. Получить индекс помощи,
нажав {F1} находясь на любом экране помощи.

     
     
     
                          ЭКРАНЫ ПОМОЩИ
-----------------------------------------------------------------

     Экраны помощи системы Paradox делятся на три области:
     
  -  Заголовок, который показывает, какую информацию содержит эк-
     ран помощи.
     
  -  Меню, которое позволяет вам выбрать другие, поясняющие,  эк-
     раны помощи или вернуться обратно к вашей работе.
     
  -  Пояснительный текст.
                   
     Экраны помощи не повторяют всю информацию, которая содержит-
ся  в  руководствах по системе Paradox, они содержат только спра-
вочную информацию. С их помощью вы можете работать с теми частями
системы, с которыми вы пока не очень хорошо знакомы или не знако-
мы вообще.
     
                                           
     
     
                           МЕНЮ ПОМОЩИ
-----------------------------------------------------------------
     
     Обратите внимание  на  меню  системы  помощи,  расположенное
вверху  начального экрана помощи. Оно содержит список главных тем
- Basics (Основные понятия), GettingAround (Обзор), Keys (Клавиа-
тура), MenuChoices (Услуги меню), Index  (Индекс)  и  Scripts/PAL
(Скрипты/PAL), - о которых можно получить информацию. При исполь-
зовании помощи меню будет изменяться в соответствии с тем, на ка-
ком экране вы находитесь.
     Для передвижения  по системе помощи используйте обычную тех-
нику. Чтобы получить информацию по одной из  тем,  приведенных  в
меню,  выберите соответствующую услугу. Чтобы вернуться на преды-
дущий экран, нажмите {Esc}.
     
                
     
     
                          ИНДЕКС ПОМОЩИ
-----------------------------------------------------------------
     
     Индекс помощи представляет из себя список тем, информацию по
которым можно получить. Поскольку экраны помощи являются контекс-
тно-зависимыми, при нажатии {F1} (помощь) на экране появится  ин-
формация только  о  текущей  операции. Если же вы хотите получить
дополнительные сведения об этой операции или информацию по другой
теме (включая сведения о синтакисе команд и функций  языка  PAL),
нажмите клавишу {F1} повторно, чтобы получить индекс помощи.
     
                                                            
     
                  Перемещение по индексу помощи
-----------------------------------------------------------------
     
     Индекс помощи  дает  вам  доступ ко всем разделам справочной
информации из любой точки системы Paradox. Активные  записи  (те,
для которых имеются экраны помощи) отмечены слева точками.
     Записи, которые  не  являются активными, используются просто
для группирования активных записей по темам. Если вы видите перед
собой на экране индекс помощи, можете использовать для  перемеще-
ния по  нему  клавиши  {Стрелка  вверх}, {Стрелка вниз}, {PgUp} и
{PgDn}. Для выбора требуемой темы нажмите {Enter}. Вы можете так-
же использовать команды {Ctrl}{Z} (найти) и {Alt}{Z} (найти  сле-
дующее) для  быстрого нахождения любой темы в индексе помощи (бо-
лее подробно об этих командах см. в главе 8).
     Если вы хотите выйти из индекса помощи, не выбрав темы, наж-
мите {Esc}. Вы вернетесь на последний экран  помощи,  который  вы
видели перед тем, как войти в индекс.

                     ВЫХОД ИЗ СИСТЕМЫ ПОМОЩИ
-----------------------------------------------------------------

     Для того,  чтобы  покинуть  экран помощи и вернуться к вашей
работе, либо выберите услугу Paradox из меню Help,  либо  нажмите
{F2} (выполнить).
