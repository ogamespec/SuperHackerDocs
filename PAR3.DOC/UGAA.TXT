... and...  have incompatible structures
(... и... имеют несовместимые структуры)

     Вы не можете добавлять записи из одной таблицы в другую, ес-
     ли их структуры несовместимы. Более  подробно  о  добавлении
     смотрите в главе 12.
                            
                            
A desired new value must be provided to Changeto
(Для Changeto должно быть указано новое значение)

     В запросе с оператором Сhangeto (изменить на) вы должны ука-
     зать новое значение, на которое вы хотите изменить существу-
     ющие.
                  
                  
A  field  cannot contain more than one expression to be Inserted
(Поле не может содержать более одного выражения для ...вставки)

     В запросе  с  оператором Insert (вставить) может содержаться
     только одно выражение для каждого  поля.  Убедитесь,  что  в
     форме  запроса  с оператором Insert в каждом поле содержится
     не более одного выражения. Более подробно о запросах с  опе-
     ратором Insert смотрите в главе 4.
    
      
A field may contain only one Changeto expression
(Поле может содержать только одно выражение с оператором  Change-
to (изменить на))

     В запросе  с  оператором  Changeto (изменить на) каждое поле
     может содержать только одно выражение с оператором. Если  вы
     хотите  изменить  несколько  значений в одном поле, сделайте
     это за несколько отдельных шагов. Более подробно о  запросах
     с оператором Changeto смотрите главу 4.
                                 
    
A set operation cannot be included in its own grouping
(Оператор set не может быть включен                                      ?
                          
     Когда вы используете в запросе ключевое слово  set  (устано-
     вить), вы  должны  сначала описать его само, а затем указать
     оператор условия  установки (only, every или exact), который
     использует ключевое слово set. Эти два компонента - описание
     ключевого поля и выражение, которое его использует - не  мо-
     гут находиться в одной строке оператора запроса. Для получе-
     ния более подробной информации смотрите главу 4.
                                                                         
     
A value must be provided in this field
(Для этого поля должно быть указано значение)

     Поле должно  быть  заполнено  -  этого требует установленный
     контроль правильности. Вы должны заполнить это поле во время
     сеанса редактирования или ввода данных. Подробнее о  провер-
     ках правильности смотрите в главе 9.
    
    
Access denied...
(Нарушение доступа...)

     Вы увидите  сообщение, начинающееся с этих слов, если вы ра-
     ботаете в сети и получить доступ к объекту или каталогу, для
     использования которого вы не имеете достаточных прав. Напри-
     ме, вы  получите  сообщение такого типа, если вы попытаетесь
     сделать ваш рабочий каталог "только читаемым". Обратитесь  к
     вашему администратору  сети, если вы не можете сами выйти из
     создавшейся ситуации.
     
     
All fields have been placed
(Все поля уже размещены)

     При создании  формы каждое регулярное поле может быть разме-
     щено в форме только один раз. Подробнее о формах смотрите  в
     главе 11.
    

All fields in a multirecord form must be in the area to be dupli-
cated (Все поля в многозаписевой форме должны находиться в облас-
ти, которая будет дублироваться)
     
     При разработке  многозаписевых  форм  все помещаемые в форму
     поля должны быть включены в область, которая  будет  служить
     как модель  для дублирующихся записей. Убедитесь, что каждое
     поле в форме, включая поля даты и номера записи, помещены  в
     эту область  при  ее описании. Для получения более подробной
     информации смотрите главу 2 книги "Представление данных сис-
     темы Paradox".
     
     
All tables on the workspace are write-protected
(Все таблицы в рабочем поле защищены от записи)

     Вы не можете редактировать ни одну из таблиц в рабочем поле,
     поскольку все они защищены от записи.
    
    
Ambigious use of ! (inclusion operator)
(Сомнительное использование оператора включения (!))
     
     Когда вы используете оператор включения несколько раз в  од-
     ном запросе, или когда вы используете его совместно с крите-
     рием выбора, результаты выполнения запроса могут зависеть от
     порядка, в котором протекает процесс выполнения. В этих слу-
     чаях предпочтительнее  преобразовать запрос к нескольким от-
     дельным шагам так, чтобы вы имели возможность точно  указать
     искомые результаты.
                         
     
Another user has since changed...
(Другой пользователь уже изменил...)
     
     Во время  сеанса совместного редактирования ваши способность
     отменять последнее сделанное изменение во многом зависит  от
     других пользователей.  Предположим, например, что вы сделали
     изменение в какой-то записи. Потом, подумав некоторое время,
     вы решили вернуться к первоначальному значению.  Однако  тем
     временем другой пользователь уже модифицировал или даже уда-
     лил данную запись. В этом случае вы не сможете отменить вне-
     сенное вами  изменение. Для получения более подробной инфор-
     мации смотрите главу 9.
     изменение
                            
     
AS must be followed by the name for the field in the Answer
(После оператора AS в таблице answer должно следовать имя поля)
     
     Оператор AS используется для явного определения имени поля в
     таблице answer. Вы должны сразу после него указать имя,  ко-
     торое вы хотите присвоить полю. Более подробно смотрите гла-
     ву 4.
    
    
Calc expression has a type error
(Вычисляемое выражение содержит ошибку типов данных)

     В форме  запроса определено выражение, которое не может быть
     вычислено, поскольку один или несколько элементов  выражения
     представляют  собой  несовместимые  типы  данных.  Например,
     нельзя складывать числа и алфавитно-цифровые значения. Более
     подробно об этом смотрите в главе 4.
 
              
Cannot append to previous line
(Строка не может быть добавлена к предыдущей строке)

     В редакторе скриптов или генераторе отчетов нажатие  клавиши
     {Backspace}  приводит к добавлению текущей строки к предыду-
     щей, если курсор стоял в начале строки.  Ошибка  показывает,
     что для выполнения операции не хватает места.
                                
                                
Cannot change CHANGED table
(Нельзя изменять таблицу changed)

     Вы не  можете использовать оператор Changeto (изменить на) в
     запросе, чтобы изменить записи во временной таблице changed.
     Переименуйте сначала эту таблицу, а затем повторите запрос.
                                
                                
Cannot delete from the DELETED table
(Нельзя удалять из таблицы deleted)
 
     Вы не можете использовать оператор Delete (удалить) в запро-
     се, чтобы удалить записи из временной таблицы deleted. Пере-
     именуйте сначала эту таблицу, а  затем  повторите  запрос  с
     оператором Delete.
                                                     
                                                     
Cannot delete master record while detail records depend on it
(Нельзя удалять главную запись, пока от нее зависят дополняющие
записи)
     
     В связанной многотабличной форме нельзя удалять главную  за-
     пись, пока  от нее зависят связанные дополняющие записи. Это
     правило обеспечивает целостность взаимосвязи  данных.  Иначе
     может получиться,  например,  что у вас в таблице будут хра-
     ниться заказы несуществующих заказчиков. Вы  должны  сначала
     удалить все  связанные с главной записью дополняющие записи,
     а потом уже сможете удалить главную.

                                         
Cannot get private directory, trying current directory
(Нет доступа к личному каталогу, пробуется текущий каталог)
     
     Личный каталог, который указан в  вашем  файле  PARADOX.CFG,
     либо не  существует,  либо используется другим пользователем
     сети, либо вы не имеете достаточных прав для доступа к нему.
     Система Paradox попытается сделать личным каталог, из  кото-
     рого вы ее запускали. Вы должны проконсультироваться у адми-
     нистратора сети,  чтобы  убедиться, что ваш файл PARADOX.CFG
     специфицирован правильно. Более подробно смотрите "Руководс-
     тво администратора сети".
     
                                     
Cannot insert into INSERTED table
(Нельзя вставлять в таблицу inserted)

     Вы не  можете использовать оператор Insert (вставить) в зап-
     росе, чтобы  вставить  новые  записи  во  временную  таблицу
     inserted. Переименуйте сначала эту таблицу, а затем повтори-
     те запрос с оператором Insert к "новой" таблице.
                                     
                                     
Cannot place duplicated area on top of fields placed on the form
(Нельзя помещать дублирующуюся область на верх полей, помещенных
в форму)
     
     В многозаписевой форме все поля должны быть помещены  в  об-
     ласть, которая  служит как модель для дублирующихся записей.
     Сама эта область не может содержать (или быть  помещена  на)
     верх любого  реального  поля  или  литералов. Более подробно
     смотрите главу  XX  книги  "Представление   данных   системы
     Paradox".
     
     
Cannot place fields or literals in the duplicated area of a  mul-
tirecord  form (Нельзя помещать поля или литералы в дублирующуюся
область многозаписевой формы)
     
     В многозаписевой форме все поля должны быть помещены  в  об-
     ласть, которая  служит как модель для дублирующихся записей.
     Сама эта область не может содержать (или быть  помещена  на)
     верх любого  реального  поля  или  литералов. Более подробно
     смотрите главу  XX  книги  "Представление   данных   системы
     Paradox".
     
     
Cannot record during script play
(Нельзя записывать во время выполнения скрипта)
     
     Вы не  можете начать запись нового скрипта во время выполне-
     ния существующего. Прервите выполнение текущего  скрипта,  а
     затем начните записывать новый.
                                
     
Cannot split field during resize
(При изменении размера не нельзя расщеплять поле)
     
     Нельзя изменять  размер  столбца в описании отчета, если при
     этом расщепляется значение поля. Установите курсор за преде-
     лы поля, прежде чем изменять ширину столбца.
             
  
Cannot toggle to linked form...
(Нельзя переключать на связанную форму...)
     
     Вы начали сеанс редактирования (совместного  редактирования)
     и не можете переключаться на многотабличную форму, не закон-
     чив сначала сеанс редактирования. Кроме того, если в рабочем
     поле находятся  несколько  образов  одной таблицы, вы должны
     удалить их все, кроме одной, перед тем, как переключаться на
     многотабличную форму. Это правило обеспечивает целостность и
     согласованность данных. По этой же причине, если вы начинае-
     те сеанс редактирования (совместного  редактирования)  через
     многотабличную форму,  вы не никаким образом сможете вносить
     изменения в таблицу, включенную в форму, не закончив сначала
     сеанс редактирования.
     
     
Cannot use LIKE  with  patterns.  Use  patterns  only  for  exact
matches only  (Нельзя использовать оператор Like с шаблонами. Ис-
пользуйте шаблоны только нахождения точного совпадения)
     
     В запросе оператор Like может использоваться только вместе с
     точными литералами или с элементами примеров. Условия в фор-
     ме типа like ..s..n в общем случае не  имеют  смысла.  Более
     подробно смотрите в главе 4.
     
     
Can't access...script
(Нет доступа к скрипту)
     
     В сети  вы не можете выполнять скрипт, если другой пользова-
     тель его редактирует.
         

Can't start Paradox: file limit in CONFIG.SYS too low
(Невозможно запустить систему Paradox: предел числа  файлов,  за-
данный в файле config.sys, слишком мал)
     
     Для запуска  системы  Paradox в корневом каталоге на жестком
     диске, с которого происходит загрузка операционной  системы,
     должен  существовать  файл  установки системных параметров с
     именем CONFIG.SYS с параметром Files  =  20.  В  большинстве
     случаев этот файл записывается во время установки на рабочий
     Системный диск автоматически. Это сообщение появляется в од-
     ном из следующих случаев: вы не  перезагрузили  операционную
     систему  с  диска, содержащего файл CONFIG.SYS, или параметр
     Files установлен неправильно.
     
     
Can't start Paradox: not enough memory
(Невозможно запустить систему Paradox: не хватает памяти)
  
     Для запуска системы на автономном компьютере требуется мини-
     мум  512  Кбайт  оперативной  памяти, а при запуске в сети -
     640. В зависимости от типа компьютера  вы  можете  увеличить
     свободную  память, уменьшая размер RAM-диска или спулера пе-
     чати, или завершить некоторые резидентные программы. Если ни
     одно из этих решений не привело к успеху,  необходимо  доба-
     вить  плату  расширения  памяти  в компьютер (Rampage Board,
     Above Board и т.д.)
      
Can't start Paradox: total number of authorized  users  has  been
reached  (Невозможно запустить  систему Paradox: предельное число
пользователей, могущих работать в сети, уже достигнуто)
     
     Количество пользователей,  могущих одновременно работающих в
     сети с системой Paradox, ограничено числом, заданным  в  ли-
     цензионном  соглашении  для  вашей сети. Если вы увидите это
     сообщение, то вам придется подождать, пока кто-нибудь из ра-
     ботающих не закончит сеанс работы с системой, прежде чем  вы
     сможете  начать работать в сети. Если вы часто получаете та-
     кое сообщение, то можете обратиться к  администратору  сети,
     чтобы  он  увеличил  число санкционированных пользователей с
     помощью покупки дополнительных копий системы Paradox или до-
     бавлением системы Paradox LAN Pack.
     
     
Delete files if possible but NOT ACTIVE PARADOX OBJECTS  OR  .LCK
FILES (Уничтожьте файлы, если возможно, но только не активные об-
ъекты и не .lck-файлы (файлы блокировок))
     
     Если вы не можете завершить операцию  из-за  того,  что  нет
     места на диске, Paradox предоставляет вам - путем временного
     выхода в  DOS  -  возможность уничтожить некоторые файлы. Вы
     должны уничтожить как можно больше файлов, но кроме тех фай-
     лов, которые содержат активные объекты системы Paradox (фай-
     лы с расширениями .DB, .Fn, .Rn, .G, .PX, .Xnn или .Ynn) или
     сетевые блокировки  (файлы  с  расширением  .LCK). Вы можете
     также увидеть некоторые временные объекты системы Paradox  с
     именами типа  ZZZ  или  ZXZZ;  их также не нужно уничтожать.
     Кроме того, вы не должны удалять любые системные фалы (файлы
     с именами PARADOX3.) Для получения более  полной  информации
     обратитесь к главе 14.
     
     
Detail table cannot be edited when corresponding master record is
blank  (Нельзя редактировать дополняющую таблицу, если соответст-
вующая ей главная запись является пустой)
     
     В связанной  многотабличной  форме  Paradox  не позволит вам
     вводить или изменять дополняющие записи, если текущая  глав-
     ная запись  имеет пустое значение ключа связи. Это предохра-
     няет целостность взаимосвязи,  иначе  вы,  например,  будете
     иметь заказы  несуществующего  заказчика.  Вы должны сначала
     специфицировать главную запись, а затем уже работать  с  до-
     полняющими ее.

(Не удовлетворяет правилам задания шаблонов языка PAL)

     задаваться по правилам, определенным в языке PAL.  Подробнее
     о  шаблонах  смотрите в главе 5 "Руководства пользователя по
     языку PAL".

     дисковода с таким именем нет в системе.

(Встроенная форма должна иметь только несвязные ключевые поля)        ?

(Поля в запросе с оператором Find нельзя помечать "галочкой")

Find can be used in only one query form at a time

(Не найден нужный драйвер  экрана,  необходимый  для  отображения
графика)
     
     Для того, чтобы выдать график на экран, в каталоге, содержа-
     щем файлы системы  Paradox,  должен  находиться  специальный
     файл, называемый драйвером экрана. Эти драйверы имеют расши-
     рение .BGI.  Если  вы  получили это сообщение, то, возможно,
     некорректно установлена  система  Paradox или нужно изменить
     путь доступа. Если вы работаете в сети, обратитесь для реше-
     ния проблемы к администратору сети.
     
     
Unable to find specified font file
(Не найден указанный файл шрифта)
     
     Различные файлы шрифтов необходимы для отображения различных
     стилей заголовков графиков. Эти файлы имеют расширение .CHR.
     Если вы получили это сообщение,  то,  возможно,  некорректно
     установлена система Paradox или нужно изменить путь доступа.
     Если  вы работаете в сети, обратитесь для решения проблемы к
     администратору сети.
     
     
Unexpected condition...
(Непредвиденное условие...)
     
     Система Paradox обнаружила непредвиденное условие, при кото-
     ром она не может продолжить выполнение  операции.  Например,
     превышено число  файлов в каталоге, разрешенное DOS. В таких
     случаях происходит выход в DOS. Для получения более  подроб-
     ной информации смотрите главу 15.
     
     
Warning! Table may be demaged! (any key to continue)
(Внимание! Таблица может быть разрушена (для продолжения  нажмите
любую  клавишу))
     
     Система Paradox обнаружила ситуацию, при которой таблица мо-
     жет быть разрушена. Вы не сможете работать с таблицей без ее
     восстановления.  Для  получения  более  подробной информации
     смотрите главу 15.
