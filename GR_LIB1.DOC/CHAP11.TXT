
11.  Фрактальная графика в режиме IBM 19
-----------------------------------------------------------------
    
    На диске,  поставляемом с  данной  книгой,  находится  мощная
программа для   создания   многоцветной   фрактальной  графики  в
соответствии с формулами,  которые разработал математик Benoit B.
Mandelbrot.
    
11.1  Создание базового файла фрактальной графики
-----------------------------------------------------------------
    
    Следующая программа,  BMANDE10.PAS,  создает   файл   рисунка
FRACTAL.PIC, который   содержит  фрактальную  графику.  Этот  тип
рисунка иногда называют "Apple Man"  ("Яблочный  человек")  из-за
форм, предлагаемых  фрактальной  графикой.  Эта  первая  картинка
будет служить графической базой для следующей программы.
    
    Листинг BMANDE10.PAS находится на программной дискете.
    
11.2  Программа BMANLED.PAS
-----------------------------------------------------------------
    
    Данная программа  работает  в режиме VGA/MCGA 19 и может быть
скомпилирована в Turbo Pascal версии 4.0 и старше.
    
    Перед выполнением программы файл FRACTAL.PIC должен уже  быть
создан программой   BMANDE10.PAS  (или  BMANDE10.EXE).  Затем  вы
можете выполнить  программу  BMANDEL.EXE,  также  находящуюся  на
диске, поставляемом с данной книгой.
    
    Стрелками выберите  точку  на  рисунке  Fractal.  При нажатии
<Enter> программа запросит у вас новое имя  файла  и  расширение.
Затем вы   можете  увеличить  часть  рисунка.  Увеличенная  часть
рисунка окружена рамкой.
    
    Время, уходящее на выполнение увеличения рисунка,  зависит от
выбранной части.  Ключевым фактором здесь является число итераций
цикла программы.  С новыми файлами,  созданными этой  программой,
можно затем   работать   при   помощи   расширенных  команд  DOS,
представленных в Главе 4.
